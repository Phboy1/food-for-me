# Welcome to Food for Me

A website that uses an API to get foods and allows me to track what I've eaten and what I've rated it.

This project was created for #Horizons

# What is it?

This is a website that I created that can store when I've tried a new food, so I can store it for myself and remember what I've eaten, what I've liked and what I haven't liked.

For me, it's a personal food tracker for every time I've tried something new. In my website, I can log it, and rate it based on what I thought. Over time this turns into a searchable library of every dished I've every tried so I never have to guess whether I liked a food or not.

# How it Works

This a website that is linked to "TheMealDB" to get access to foods, images and names of all of the foods which are getting imported into a react.js app. From there, I can save it to my personal log along with a wrating and any notes I wanted to add. Everything is build in React.

# What I learnt

I learned alot about React.js, specifically, context windows, compenents, and lots of the weird syntax in javascript. State and context were probably the trickest part to wrap my mind around. Understanding how the date flows through a React app, and when to use local state vs context was very confusing to me. It took many attempts and google searches for me to actually understand how these concepts work. Overall, I felt like it was really cool to build something completely from scratch for something that I would actually use in my daily life. Hopefully I'll be eating alot of food in the future.

# AI USAGE

I did use ai in this project, but only for the css styling which is just a tedious endeavour and I don't really have an artistic eye. Additionally, sometimes I would run into a merge conflict and I would ask AI for help. Everything else was by myself including learning how to use react and creating this README.md

# Motivation

I found this app called Beli which tracked which restaurants I've gone to and how I've rated them in the past. I wanted to put a spin on this and create a library all the foods that I've tried before. This is so I never have to sit there scratching my head wondering if I liked the food that I picked or not.
